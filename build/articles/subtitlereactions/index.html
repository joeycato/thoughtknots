Subtitle Reactions!

31 August 2017

Netflix Hack Day has arrived again! For this experimental hack, we decided to add multiple visualizations to Netflix subtitles during playback.

Basically, each time a subtitle changes on-screen, we add the following overlays:

  • Image stickers ( sourced by GIPHY API searchâ€™s top result )
  • Emojis ( based on moji-translate library )
  • Mood colorization ( based on AFINN sentiment library )
  • As a bonus, if we detect song lyrics a disco ball drops down from the top of the screen

Special Thanks to Sai Shenoy and Jacklyn Ng for helping out!
